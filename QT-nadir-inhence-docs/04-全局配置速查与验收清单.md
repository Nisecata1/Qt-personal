# QtScrcpy 全局配置速查与验收清单

## 1. 用途

本文件是“跨模块速查页”，用于快速定位参数、生效时机和验收步骤。实现细节请跳转对应模块文档。

## 2. 文档入口

1. 视角模式稳定：`01-输入链路-视角模式稳定性手册.md`
2. 方向盘回正：`02-输入链路-方向盘回正配置手册.md`
3. 远端光标与 server：`03-服务端-远端光标渲染与构建手册.md`
4. 目录规范：`00-文档导航与模块说明.md`

## 3. 参数速查（核心）

| 文件           | 分组       | 参数                           | 建议值   | 生效说明                 |
| -------------- | ---------- | ------------------------------ | -------- | ------------------------ |
| `userdata.ini` | `[common]` | `MouseSmoothFrames`            | `8`      | 视角平滑窗口             |
| `userdata.ini` | `[common]` | `SteerWheelRecoverySpeed`      | `0.45`   | 方向盘回正速度（热加载） |
| `userdata.ini` | `[common]` | `SteerWheelRecoveryNoise`      | `0.0020` | 中段噪声（热加载）       |
| `userdata.ini` | `[common]` | `SteerWheelRecoveryFinalNoise` | `0.0004` | 终点微噪（热加载）       |
| `config.ini`   | `[common]` | `RemoteCursorEnabled`          | `true`   | 远端光标总开关           |
| `config.ini`   | `[common]` | `CursorSizePx`                 | `64`     | 光标大小，建议 `8..128`  |

## 4. 推荐最小配置模板

### 4.1 输入链路（userdata.ini）

```ini
[common]
MouseSmoothFrames=8
SteerWheelRecoverySpeed=0.45
SteerWheelRecoveryNoise=0.0020
SteerWheelRecoveryFinalNoise=0.0004
```

### 4.2 光标渲染（config.ini）

```ini
[common]
RemoteCursorEnabled=true
CursorSizePx=64
```

## 5. 联合验收清单

1. 退出/重进视角模式 10 次，无首跳、无残留触控。
2. `W+A -> W`、`W+D -> D` 等退化场景可平滑回正。
3. Normal 模式可见远端光标；切入 Game 模式后可隐藏。
4. 光标大小配置调整后生效。
5. 常规点击、拖拽、转向行为均正常。

## 6. 维护规则

1. 本文件仅维护“速查与入口”，不复制模块实现细节。
2. 参数新增时，优先更新对应模块文档，再回填本速查页。
3. 若发现重复内容，统一以模块文档为准并删除重复段落。
