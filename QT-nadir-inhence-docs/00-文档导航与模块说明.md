# QtScrcpy 文档导航与模块说明

## 1. 目录目标
本目录用于沉淀 QtScrcpy 个人改造文档，按模块拆分，避免同一内容在多份手册重复维护。

## 2. 模块文档清单

1. `01-输入链路-视角模式稳定性手册.md`  
   负责：相对视角模式（鼠标转向）稳定性修复、首帧策略、会话重置、验收步骤。
2. `02-输入链路-方向盘回正配置手册.md`  
   负责：SteerWheel 向量回正机制、`userdata.ini` 三个回正参数、调优与验收。
3. `03-服务端-远端光标渲染与构建手册.md`  
   负责：`0x80/0x81` 协议、SurfaceControl 光标层、server 构建与替换流程。
4. `04-全局配置速查与验收清单.md`  
   负责：跨模块参数速查、生效时机速查、联合验收流程（不展开实现细节）。

## 3. 去重策略

1. 实现细节只写在对应模块文档，其他文档通过“跳转链接 + 摘要”引用。
2. `04` 不复制代码级解释，仅做总览和入口。
3. 新增功能时先判断归属模块，再更新对应文档与 `04` 速查。

## 4. 命名规范

1. 使用前缀编号确保阅读顺序：`00/01/02/03/04`。
2. 文件名格式：`序号-模块-主题.md`。
3. 标题格式：`QtScrcpy + 模块名称 + 手册/速查`。
